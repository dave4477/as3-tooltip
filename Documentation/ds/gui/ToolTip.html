Packageds.gui
Classpublic class ToolTip
InheritanceToolTip flash.display.Sprite

The ToolTip class defines static functions to generate a generic tooltip. The tooltip is always added to the stage itself, and is typically called in the mouseover's event listener function for a displayobject. See changelog on http://detailed-simplicity.xs4all.nl/ds2/
